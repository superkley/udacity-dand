In a recent data analysis project, we analyzed the Titanic data set rigorously. The main outcome was the answer to the crusial question - what factors made people more likely to survive?

By juggling with the three biggest influencers on the survival chance namely 'Sex', 'Ticket Class' (Pclass) and 'Age Group', this web page acts as a playground for visual data storytelling utilizing the findings and exploring the different relationships between the three variables.

In the next step let's check out the interactive visualization to clear the facts up. It's basicly a clean and simple bar chart with a legend, data labels and axis titles. Rather than showing static charts and putting fixed explanations you have free control over all the combinations possible with the three categories and you'll be inspired to create your own interpretations of which and why certain groups could survive or not.

The last step is for digging into the numbers. A dynamic table with simplified Titanic data is displayed there. You have the options to sort and filter the records.

P.S. If you're interested in the original data. The Titanic data is provided here for you as a download link. It contains demographics and passenger information from 891 of the 2224 passengers and crew on board the Titanic from which we derived the data set visualized on this page. (Note: only passengers with a registered age are analyzed here.) A full description of the original dataset can be found on the Kaggle website, where also the data was obtained.
